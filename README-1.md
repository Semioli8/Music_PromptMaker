# 🎵 Music Prompt Maker

A free, mobile-friendly web app that generates AI-ready prompts for [Suno](https://suno.com) music generation — outputting separate **Lyrics** and **Styles** code blocks you can paste directly into ChatGPT or any AI tool.

## Features

- 🎛 **26 music genres** — Techno, Pop, Hip-Hop, Synthwave, Arabesk and more
- 🌍 **TR / EN interface** — switch languages on the fly
- 🌙 **Dark / Light theme** toggle
- 🎲 **Random mode** — fills all fields instantly
- 🎤 **Instrumental toggle** — skips lyrics, generates structure tags only
- 📋 **One-tap copy** — copies the full prompt in one click
- 📱 **Mobile-first** — works as a PWA when added to home screen

## How to Use

1. Select your genre, language, energy, tempo, mood and other settings
2. (Optional) Add extra notes like `deep bass, cinematic intro`
3. Hit **⚡ PROMPT ÜRET / GENERATE PROMPT**
4. Copy the output and paste it into ChatGPT, Claude or any AI
5. The AI will return two code blocks:
   - ` ```lyrics ``` ` → paste into Suno's **Lyrics** field
   - ` ```styles ``` ` → paste into Suno's **Style of Music** field
